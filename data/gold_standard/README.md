# PRISM-LLM Gold Standard Dataset

## Contents (released upon publication)
- `extractions/` — 200 papers with expert PICO-Perio extractions
- `rob_assessments/` — RoB 2 / ROBINS-I by Cochrane-trained reviewers
- `screening_labels/` — Dual-reviewer consensus include/exclude labels
- `synthesis_scores/` — Expert panel composite quality scores

## Annotation Protocol
- 3 calibrated periodontists + 2 evidence synthesis methodologists
- Inter-annotator agreement: kappa >= 0.85
- Discrepancies resolved by consensus meeting
